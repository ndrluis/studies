# Studies

This repository is a place for my personal programming studies.

- `elixir/parquet-reader` - an exploration of reading Parquet files

## How I Use AI Here

I use AI in this repository as a study pair.

That means I may ask AI to explain concepts, discuss tradeoffs, review my understanding, point me toward things to inspect, or help me reason through a problem. I do not use AI as the author of the code or as the decision-maker for what to build next.

All code in this repository is written by me. The definitions of the next steps, the learning goals, and the direction of each study are also my own.

## Notes

This is a playground, so I do not care about keeping a clean commit history.
Commit messages may be rough, temporary, or only meaningful to me at the moment I wrote them.

Issues and pull requests are closed. If you want to suggest something, point out that I am doing something wrong, or discuss one of the studies, please send me an email instead.

## Intent

The main goal is learning through deliberate practice. Some code may be incomplete, experimental, or intentionally small because the point is to understand the underlying ideas before expanding the implementation.
